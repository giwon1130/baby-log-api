# 📚 베이비로그 (API) 문서

이 폴더는 **사람용** 문서야. 자동화 에이전트(Claude 등) 용 노트는 저장소 루트의
[`AGENTS.md`](../AGENTS.md) 에 있어.

이 저장소는 **백엔드 (Spring Boot 3.3 / Kotlin / PostgreSQL)** 야.
앱 저장소는 [`baby-log-app/docs/`](https://github.com/giwon1130/baby-log-app/tree/main/docs).

## SoT 정책

- **API 명세 / 데이터 모델 / 배포 절차 / 환경변수** → 이쪽 (서버) 저장소
- **제품/디자인/사용자 시나리오/UX** → 앱 저장소
- 양쪽이 같은 토픽 다루면 cross-link
