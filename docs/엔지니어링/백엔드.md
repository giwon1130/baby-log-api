```kotlin
jdbcTemplate.execute("""
    create table if not exists bl_medications (
        id varchar(36) primary key,
        baby_id varchar(36) not null references bl_babies(id),
        taken_at timestamptz not null default now(),
        name varchar(100) not null,
        dose_mg double precision,

""".trimIndent())

```

#### C. Service
```kotlin
@Service
class MedicationService(private val jdbc: JdbcTemplate) {
    fun record(babyId: String, request: RecordMedicationRequest): MedicationResponse {
        val id = UUID.randomUUID().toString()
        jdbc.update(
            "insert into bl_medications (id, baby_id, taken_at, name, dose_mg, note) values (?, ?, ?, ?, ?, ?)",
            id, babyId, request.takenAt ?: Instant.now(),
            request.name, request.doseMg, request.note,
        )
        return get(id)
    }

    fun list(babyId: String, limit: Int = 50): List<MedicationResponse> {
        return jdbc.query(
            "select * from bl_medications where baby_id = ? order by taken_at desc limit ?",
            { rs, _ -> mapRow(rs) },
            babyId, limit,
        )
    }

    private fun mapRow(rs: ResultSet) = MedicationResponse(
        id = rs.getString("id"),
        babyId = rs.getString("baby_id"),
        takenAt = rs.getTimestamp("taken_at").toInstant().toString(),
        name = rs.getString("name"),
        doseMg = (rs.getObject("dose_mg") as? Double),
        note = rs.getString("note"),
    )
}

data class RecordMedicationRequest(
    val takenAt: Instant? = null,
    val name: String,
    val doseMg: Double? = null,
    val note: String = "",
)

data class MedicationResponse(
    val id: String,
    val babyId: String,
    val takenAt: String,
    val name: String,
    val doseMg: Double?,
    val note: String,
)
```

#### D. Controller
```kotlin
@RestController
@RequestMapping("/api/v1/babies/{babyId}/medications")
class MedicationController(private val service: MedicationService) {

    @PostMapping
    fun record(
        @PathVariable babyId: String,
        @RequestBody request: RecordMedicationRequest,
    ): MedicationResponse = service.record(babyId, request)

    @GetMapping
    fun list(
        @PathVariable babyId: String,
        @RequestParam(defaultValue = "50") limit: Int,
    ): List<MedicationResponse> = service.list(babyId, limit)
}
```

### 2. 기존 도메인 컬럼 추가

스키마 마이그레이션:
```kotlin
// DatabaseConfig.SchemaInitializer.init() 끝에
jdbcTemplate.execute("alter table bl_feeds add column if not exists left_minutes integer")
```

이미 있는 컬럼이면 무시됨. 안전.

`Service` 의 INSERT/SELECT 쿼리에 컬럼 추가, DTO에 필드 추가.

### 3. 컬럼 삭제 / 타입 변경

`if not exists` 같은 안전 DDL 없음. 수동 작업:
1. Railway DB 콘솔 접속 (`railway connect postgres`)
2. SQL 직접 실행
3. 코드에서 해당 컬럼 참조 제거

운영 DB 변경은 항상 백업 후. 이력 [`의사결정로그.md`](의사결정로그.md) 에 기록.

## 코드 스타일

- Kotlin 1.9 기능 사용 — `data class`, scope functions, sealed class 등
- 명시적 타입 (`val x: Int = 5`) — 추론 가능하면 생략
- DTO 는 `data class`
- 옵셔널 페이로드 — nullable + 기본값
- `JdbcTemplate.update` / `query` — 직접 사용. Spring Data JDBC 도입 검토 보류

## 에러 처리

현재: `IllegalArgumentException` / `NoSuchElementException` 던지면 Spring이 4xx로 변환.
별도 ExceptionHandler 미설치.

향후 표준화 시:
```kotlin
@RestControllerAdvice
class GlobalExceptionHandler {
    @ExceptionHandler(IllegalArgumentException::class)
    fun badRequest(e: IllegalArgumentException) =
        ResponseEntity.badRequest().body(mapOf("error" to e.message))
}
```

## 로깅

`println()` 또는 `logger.info("...")`. 정형화 안 됨. 운영 시 Railway 대시보드 로그.

PII 로그 금지 — 아기 이름, 메모 등은 찍지 않기.
